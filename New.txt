 Role of C in the Unix C Language

    The kernel (core of Linux) is primarily written in C.
        Why C?

            Portability: Can run on different hardware architectures.
            Efficiency: Produces fast, low-level machine code.
            Control: Allows direct memory and hardware manipulation.
2. Assembly Language

       Only a tiny part of Linux uses assembly.
       Usually for bootstrapping the system, handling hardware interrupts, or CPU-specific instructions.
3. Summary

       Linux Kernel = Mostly C + Some Assembly
       Most Linux tools, utilities, and applications can be written in C, C++, Python, Shell scripts, etc., but the core OS is C-based.s
